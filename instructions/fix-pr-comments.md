comment-triage.md の「対応必須」コメントに対してコード修正を行え。

**やること:**
1. `comment-triage.md` を読み込み、「対応必須 (required)」のコメント一覧を取得する
2. 各コメントについて:
   - 指摘対象のファイルと行を特定する
   - 現在のコードを読み込み、指摘内容を理解する
   - 適切な修正を実施する
3. 修正後、関連するテストを更新・追加する
4. 品質ゲートを実行する:
   - lint と test の両方を実行し成功を確認する
   - いずれかが失敗した場合は修正を続ける
5. 修正内容をコミットする:
   - コミットメッセージ: `fix: address PR review comments`

**重要:**
- バッチ外のタスクやスコープ外の変更は行わない
- 修正が既存機能に影響する場合はリグレッションテストを実行する
- 修正不可能な指摘（アーキテクチャレベルの変更要求等）はABORTし、確認を求める
