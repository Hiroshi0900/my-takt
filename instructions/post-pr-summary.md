08-pr-preparation.md のセルフレビュー結果をPRコメントとして投稿せよ。

**やること:**
1. `08-pr-preparation.md` を読み込み、セルフレビュー結果を確認する
2. `09-pr-created.md` を読み込み、PR番号を取得する
3. セルフレビューで指摘がある場合のみ、PRコメントを投稿する:
   ```
   gh pr comment <number> --body "<セルフレビューサマリー>"
   ```
4. コメント本文のフォーマット:
   ```
   ## Self-Review Summary

   以下の点が検出されました。レビュー時にご確認ください。

   | Severity | 内容 | ファイル |
   |----------|------|---------|
   | Warning | {指摘内容} | {ファイルパス} |
   ```
5. 指摘がない場合はコメント投稿をスキップする

**重要:**
- このmovementは非ブロッキング: 投稿失敗してもCOMPLETEに進む
- Critical指摘はこの段階では存在しない（prepare-prで弾かれるため）
- コメントは情報提供目的であり、ブロッカーではない
